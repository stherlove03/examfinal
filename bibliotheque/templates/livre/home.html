Bibliotheque

Le projet de bibliothèque est une application web conçue pour gérer les informations relatives aux livres dans une bibliothèque. Cette application permet aux utilisateurs de consulter, ajouter, modifier et supprimer des livres



L'application comprend plusieurs fonctionnalités clés, notamment :

Les livres Permet aux administrateurs d'ajouter de nouveaux livres, de mettre à jour leurs informations

Interface d'Administration : Les administrateurs ont accès à une interface d'administration dédiée qui leur permet de gérer tous les aspects de l'application, y compris les utilisateurs, les groupes, les autorisations, etc.
